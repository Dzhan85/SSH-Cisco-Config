
Script for Access and Configure Cisco LAN Devices.


Before use:
Python Version 2
Needed an hostfile with the name of the device (EX:192.68.2.250), this file must be an text file or a file without extension.
On module sshing.py change lines 8 & 9 to correct user and password.
On Main.py change line 32 array commands to your suitable commands.

Modules used:
re
sys
os
pexpect
from pexpect pxssh

To use:
On shell or Bash run: python Main.py hostfile
